# Electricity Usage Pattern Classification

A time-series machine learning project that classifies electricity consumption behavior into Normal, Spike, and Rising patterns using classical ML techniques.
---

## Features
- Time-series feature extraction
- Random Forest classification
- Synthetic data generation
- Fast and lightweight

---

## Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn

---

## How to Run
```bash
pip install numpy pandas scikit-learn joblib
python generate_data.py
python train_model.py
python predict.py
```
---

## Output
-Usage pattern prediction
-Classification accuracy
